# 文体・記法ガイド

## 基本文体

### 視点
- **三人称一元視点**（蓮の視点）
- 蓮の知覚・感情のみ描写する。他キャラの内心は行動や表情から推測する形で
- 例: ×「ルナは悲しかった」→ ○「ルナの表情が一瞬翳ったように見えた」

### 文体トーン
- ライトノベル寄りの読みやすい文体
- 硬すぎず、砕けすぎず
- 地の文は標準的な書き言葉。ただし一人称的な蓮の心情描写を適度に混ぜる
- 例:「まずい。そう思った瞬間、体が勝手に動いていた」

### 文長
- 一文は40字以内を目安。長くても60字まで
- 短文を連ねてテンポを出す。特に戦闘シーンや緊迫場面
- 描写シーンではやや長めの文も可

### 文末バリエーション
- **同一文末の3回以上連続を禁止**する
- ～た / ～ている / ～ていた / ～ない / 体言止め / ダッシュ止め（——）を意識的に混ぜる
- 例:
  - ×「剣を抜いた。構えを取った。相手を睨んだ。」（3連続「～た」）
  - ○「剣を抜いた。低い構え。相手の目を射抜くように——」

### 段落の長さ
- 地の文1段落は **2〜4文** を目安とする
- 5文以上になる場合は分割を検討する
- 会話が連続する箇所では、3往復以上続く場合は地の文（動作・表情・情景）を挟んでリズムを保つ

## 「見せる」原則（Show, Don't Tell）

すべての描写において「説明」ではなく「提示」を基本とする。読者が自ら感じ取れる描写を目指す。

### 感情を見せる
- 感情名（嬉しい・悲しい・怒り等）を直接書かず、身体反応・行動・五感で示す
- ○「視界が滲んだ。慌てて袖で目元を拭う」
- ×「蓮は悲しくなった」
- ○「指先が震えている。握り直しても、震えは止まらなかった」
- ×「蓮は恐怖を感じた」

### 世界観・設定を見せる
- 「この世界では〜という仕組みがある」と説明せず、キャラの行動・会話・日常の中で自然に伝える
- ○「パン屋の親父が、代金の代わりに昨夜見た夢を差し出した。それがこの街の普通だった」
- ×「この世界では記憶が通貨として使われている」

### 能力・強さを見せる
- 「彼は強い」「すごい魔法」と書かず、具体的な描写で読者に実感させる
- ○「剣が閃いた——次の瞬間、三体の灰獣が同時に崩れ落ちた。ガルドは息ひとつ乱していない」
- ×「ガルドは凄腕の剣士だった」

### 関係性を見せる
- 「二人は仲が良い」「信頼している」と直接書かず、やり取りや行動で関係性を浮かび上がらせる
- ○「蓮が黙って差し出した水筒を、ルナは何も言わず受け取った」
- ×「二人の間には信頼関係が築かれていた」

## 記法ルール

### 改行・空行ルール
- **セリフの前後に空行を1つ**入れる
- 地の文同士が続く場合は空行なし（段落が変わる場合は空行1つ）
- 例:
  ```
  　霧の中から一人の少女が現れた。銀色の髪が風に揺れている。

  「……生きてるの」

  　声に感情はなかった。確認するような、それだけの言葉。

  「な、んとか……」

  　掠れた声でそう答えるのが精一杯だった。
  ```

### 会話文
- 「」で囲む。二重カギカッコ『』は書籍名・固有名詞に使用
- 会話文の末尾に「と〇〇は言った」を毎回つけない。動作や描写で話者を示す
- 例:
  ```
  「ここは危険だ」

  　ガルドが剣を抜きながら呟いた。その視線は霧の奥を射抜いている。

  「……分かってる」
  ```

### 心内語
- 地の文に溶け込ませる
- 例: ここを知っている。そんな確信が、理由もなく胸に湧いた。

### シーン転換
- 空行 + アスタリスク3つで区切る
- ```
  （シーン1の末尾）

  　＊　＊　＊

  （シーン2の冒頭）
  ```

### ルビ・ふりがな
- ルビは極力使わない。読みやすさを優先する
- 記法: 漢字《ふりがな》
- スキル名・術式名はカタカナ表記のみ（例: プロテクト、メモリア・クラフト）
- 組織名・造語にはルビ不要（例: 記憶管理局、灰人）
- 人名・地名のみ初出時にルビ（例: 灰ノ辻《はいのつじ》）
- 2回目以降は全てルビなし

### 固有名詞の初出ルール
- キャラクターの名前は、作中で名乗る・判明する場面を経てから地の文で使用する
- 判明前は「少年」「少女」「男」など一般呼称を使う
- 前話で名前が判明済みなら次話冒頭から使用してよい
- 新キャラ登場時は「いつ・どう名前が読者に開示されるか」を意識する
- アイテム・地名・術式名・組織名についても同様に、初出時は文脈で何であるか伝わる描写を添える

### 強調
- 傍点の代わりに「・」を文字の間に入れない。代わりに文脈で強調する
- ダッシュ（——）を使った間で強調可

### 読点
- リズムの道具として **最小限** に使う
- 1文に読点2つまでを目安とする。3つ以上になる場合は文を分割する
- ○「霧の中を、蓮は走った」
- ×「霧の中を、蓮は、剣を握りしめながら、走った」（→ 文を分割）

## 描写のガイドライン

### 五感描写
- 毎シーンで最低2つの感覚を使う
- 視覚に偏りがち。意識的に音・匂い・触感・温度を入れる
- 例:「鉄錆のような匂いが風に乗ってきた。剣を握る手が冷たい」

### 冒頭の吸引力
- **エピソード冒頭**は五感または感情で始める。状況説明から入らない
  - ○「灰が降っていた。音もなく、途切れることなく——」
  - ×「蓮たちは灰ノ辻を出発してから三日目を迎えていた」
- **シーン冒頭**も同様に、感覚的な描写で場所・時間を提示する（→ エピソード構成テンプレート「フック」も参照）

### 戦闘描写
- 動きのあるアクションは短文でテンポよく（→ テンポ制御も参照）
- 「発動→効果→代償」の流れを丁寧に
- 魔法使用時は「何の記憶を使ったか」を必ず描写する
- 例:「グリモワールの頁が光る。昨日見た夕焼けの記憶が、炎となって掌に宿った」

### テンポ制御（緩急）
- **静→動→静** の波を意識する。一本調子にしない
- **加速**: 短文・体言止め・会話の応酬で緊迫感を出す
- **減速**: 長めの文＋五感描写・心情描写で余韻を持たせる
- **会話のテンポ**: 会話が3往復以上続く場合は、地の文（動作・表情・情景）を挟んでリズムを保つ
- 例（加速→減速）:
  ```
  　剣が閃いた。火花。後退。間合いを詰め直す——

  　戦いが終わると、森は嘘のように静かだった。木漏れ日が、折れた剣の破片をゆっくりと照らしている。
  ```

### 感情描写
- **「見せる」原則** に従い、感情名を直接書かず身体反応や行動で示す
- 複数の感覚を重ねて感情の強度を伝える
- 例: ×「蓮は悲しくなった」→ ○「視界が滲んだ。慌てて袖で目元を拭う。指先が冷たい」
- 微細な感情の変化は、仕草の細部（指先・視線・呼吸）で表現する

### 情景描写
- 冒頭で場所・時間・天候を自然に提示
- 例:「朝靄がまだ残る石畳の通りに、パン屋の焼ける匂いが漂い始めていた」

### 比喩・直喩
- **作品主題に沿った比喩** を優先する: 記憶・喪失・灰・霧・光と影
  - ○「記憶が砂のように指の間からこぼれ落ちていく」
  - ○「灰に覆われた空のような、曇った瞳」
- **現代・地球由来の比喩は地の文でも禁止**（蓮の心内語は例外）
  - ×「スイッチが入ったように」「映画のワンシーンみたいに」
- 1シーンに凝った比喩は **1〜2個** に留める。多用すると文章が重くなる
- 直喩（〜のように、〜みたいに）と暗喩を混ぜ、単調にしない

### 反復・対句
- **意図的な反復** は効果的に使う:
  - 対比強調:「記憶を失って得たもの。記憶を得て失ったもの」
  - エピソード内の呼応: 冒頭と末尾で同じモチーフを変奏する
  - 話数をまたぐコールバック: 過去のエピソードの印象的な一節を形を変えて再登場させる
- **無意識な重複は避ける**: 近接する2文で同じ動詞・形容詞を繰り返さない
  - ×「剣を握った。強く握った」→ ○「剣を握った。指が白くなるほど——」

## 禁止事項

1. **説明口調の地の文**: 「この世界では〜という仕組みがある」のような説明は避ける。キャラの行動や会話の中で自然に伝える（→「見せる」原則: 世界観・設定、能力・強さ、関係性 を参照）
2. **過剰なモノローグ**: 心情の語りが3文以上続くとテンポが落ちる
3. **重複表現**: 「頭を頷く」「最後の最終手段」など
4. **現代語・現代比喩の混入**: 異世界の会話に「スマホ」「コンビニ」等は使わない。地の文の比喩でも「スイッチが入ったように」等の現代由来の表現は禁止（蓮の心内語は例外）（→ 比喩・直喩 を参照）
5. **ご都合展開の連発**: ピンチを安易に解決しない。代償や葛藤を伴わせる
6. **メタナラティブ参照**: 「第7話で〜」「前回の〜」など、エピソード番号や話数構造への言及は禁止。読者・キャラクターは話数を知らない。過去の出来事を参照する場合は、作中世界の具体的な描写（場所・状況・時期）で指し示す
   - ×「第7話で一瞬だけ見えた文字列」
   - ○「草原で初めて術式を試みた時、頁の隅で一瞬だけ明滅した文字列」
7. **未導入の固有名詞**: アイテム・地名・術式名・組織名を初めて本文中に登場させる際は、文脈から何であるか分かるように書く。前話までに登場済みの名詞はそのまま使用可

## 表記の正式名称

固有名詞の表記は `story/setting.md` および `story/characters.md` に定義された表記を正とする。執筆・改稿時は必ずこれらを参照し、表記を一致させること。

主な注意例:
- 灰術（アッシュ・アーツ）— ×灰の術、×アッシュアーツ
- 記憶術式（メモリア・クラフト）— ×メモリアクラフト
- 灰のグリモワール — ×グリモア
- ヴェルディア — ×ベルディア、×ヴェルデリア
- ミレーヌ — ×ミレーネ、×ミレーユ

## エピソード構成テンプレート

1. **フック**（冒頭1〜3行）: 五感や感情で読者を引き込む場面や台詞（→ 冒頭の吸引力 を参照）
2. **状況提示**（序盤）: 時間・場所・前話からの連続
3. **展開**（中盤）: 本話のメインイベント
4. **転換/深化**（中盤後半）: 予想外の展開や感情の深まり
5. **引き**（末尾）: 次話への期待を高めるフック

### 伏線の埋め方
- **描写に自然に埋め込む**: 伏線だと気づかせない形で情景・小道具・五感に仕込む
  - ○「壁の紋様の一部が、かすかに光っているように見えた——気のせいだろう」
- **キャラの反応の不自然さで暗示する**: 読者に「なぜ？」と思わせる行動・表情を挟む
  - ○「その名前を聞いた瞬間、ミレーヌの手が一瞬だけ止まった」
- **1エピソードに新規伏線は1〜2本** を目安とする。過剰に仕込むと回収が困難になる
- 伏線の回収時は、読者が「あの場面はこういう意味だったのか」と感じられる距離感を意識する
