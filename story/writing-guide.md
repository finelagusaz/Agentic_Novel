# 文体・記法ガイド

## 基本文体

### 視点
- **三人称一元視点**（蓮の視点）
- 蓮の知覚・感情のみ描写する。他キャラの内心は行動や表情から推測する形で
- 例: ×「ルナは悲しかった」→ ○「ルナの表情が一瞬翳ったように見えた」

### 文体トーン
- ライトノベル寄りの読みやすい文体
- 硬すぎず、砕けすぎず
- 地の文は標準的な書き言葉。ただし一人称的な蓮の心情描写を適度に混ぜる
- 例:「まずい。そう思った瞬間、体が勝手に動いていた」

### 文長
- 一文は40字以内を目安。長くても60字まで
- 短文を連ねてテンポを出す。特に戦闘シーンや緊迫場面
- 描写シーンではやや長めの文も可

## 記法ルール

### 改行・空行ルール
- **セリフの前後に空行を1つ**入れる
- 地の文同士が続く場合は空行なし（段落が変わる場合は空行1つ）
- 例:
  ```
  　霧の中から一人の少女が現れた。銀色の髪が風に揺れている。

  「……生きてるの」

  　声に感情はなかった。確認するような、それだけの言葉。

  「な、んとか……」

  　掠れた声でそう答えるのが精一杯だった。
  ```

### 会話文
- 「」で囲む。二重カギカッコ『』は書籍名・固有名詞に使用
- 会話文の末尾に「と〇〇は言った」を毎回つけない。動作や描写で話者を示す
- 例:
  ```
  「ここは危険だ」

  　ガルドが剣を抜きながら呟いた。その視線は霧の奥を射抜いている。

  「……分かってる」
  ```

### 心内語
- 地の文に溶け込ませる
- 例: ここを知っている。そんな確信が、理由もなく胸に湧いた。

### シーン転換
- 空行 + アスタリスク3つで区切る
- ```
  （シーン1の末尾）

  　＊　＊　＊

  （シーン2の冒頭）
  ```

### ルビ・ふりがな
- ルビは極力使わない。読みやすさを優先する
- 記法: 漢字《ふりがな》
- スキル名・術式名はカタカナ表記のみ（例: プロテクト、メモリア・クラフト）
- 組織名・造語にはルビ不要（例: 記憶管理局、灰人）
- 人名・地名のみ初出時にルビ（例: 灰ノ辻《はいのつじ》）
- 2回目以降は全てルビなし

### 固有名詞の初出ルール
- キャラクターの名前は、作中で名乗る・判明する場面を経てから地の文で使用する
- 判明前は「少年」「少女」「男」など一般呼称を使う
- 前話で名前が判明済みなら次話冒頭から使用してよい
- 新キャラ登場時は「いつ・どう名前が読者に開示されるか」を意識する
- アイテム・地名・術式名・組織名についても同様に、初出時は文脈で何であるか伝わる描写を添える

### 強調
- 傍点の代わりに「・」を文字の間に入れない。代わりに文脈で強調する
- ダッシュ（——）を使った間で強調可

## 描写のガイドライン

### 五感描写
- 毎シーンで最低2つの感覚を使う
- 視覚に偏りがち。意識的に音・匂い・触感・温度を入れる
- 例:「鉄錆のような匂いが風に乗ってきた。剣を握る手が冷たい」

### 戦闘描写
- 動きのあるアクションは短文でテンポよく
- 「発動→効果→代償」の流れを丁寧に
- 魔法使用時は「何の記憶を使ったか」を必ず描写する
- 例:「グリモワールの頁が光る。昨日見た夕焼けの記憶が、炎となって掌に宿った」

### 感情描写
- 「嬉しい」「悲しい」と直接書かず、身体反応や行動で示す
- 例: ×「蓮は悲しくなった」→ ○「視界が滲んだ。慌てて袖で目元を拭う」

### 情景描写
- 冒頭で場所・時間・天候を自然に提示
- 例:「朝靄がまだ残る石畳の通りに、パン屋の焼ける匂いが漂い始めていた」

## 禁止事項

1. **説明口調の地の文**: 「この世界では〜という仕組みがある」のような説明は避ける。キャラの行動や会話の中で自然に伝える
2. **過剰なモノローグ**: 心情の語りが3文以上続くとテンポが落ちる
3. **重複表現**: 「頭を頷く」「最後の最終手段」など
4. **現代語の混入**: 異世界の会話に「スマホ」「コンビニ」等は使わない（蓮の心内語は例外）
5. **ご都合展開の連発**: ピンチを安易に解決しない。代償や葛藤を伴わせる
6. **メタナラティブ参照**: 「第7話で〜」「前回の〜」など、エピソード番号や話数構造への言及は禁止。読者・キャラクターは話数を知らない。過去の出来事を参照する場合は、作中世界の具体的な描写（場所・状況・時期）で指し示す
   - ×「第7話で一瞬だけ見えた文字列」
   - ○「草原で初めて術式を試みた時、頁の隅で一瞬だけ明滅した文字列」
7. **未導入の固有名詞**: アイテム・地名・術式名・組織名を初めて本文中に登場させる際は、文脈から何であるか分かるように書く。前話までに登場済みの名詞はそのまま使用可

## 表記の正式名称

固有名詞の表記は `story/setting.md` および `story/characters.md` に定義された表記を正とする。執筆・改稿時は必ずこれらを参照し、表記を一致させること。

主な注意例:
- 灰術（アッシュ・アーツ）— ×灰の術、×アッシュアーツ
- 記憶術式（メモリア・クラフト）— ×メモリアクラフト
- 灰のグリモワール — ×グリモア
- ヴェルディア — ×ベルディア、×ヴェルデリア
- ミレーヌ — ×ミレーネ、×ミレーユ

## エピソード構成テンプレート

1. **フック**（冒頭1〜3行）: 読者を引き込む場面や台詞
2. **状況提示**（序盤）: 時間・場所・前話からの連続
3. **展開**（中盤）: 本話のメインイベント
4. **転換/深化**（中盤後半）: 予想外の展開や感情の深まり
5. **引き**（末尾）: 次話への期待を高めるフック
