# 全体プロット（第一部: 全20話）

## 構成

### 第一幕: 覚醒と出会い（第1話〜第5話）

**第1話「灰色の目覚め」**
- 蓮が霧の荒野で目覚める。記憶がほとんどない。手に灰のグリモワール
- 魔獣に襲われ、無意識にグリモワールで魔法を発動（代わりに「朝食の記憶」を失う）
- ルナに助けられる。ルナは蓮に興味を示し、行動を共にすることに
- 引き: ルナの手首の紋章が一瞬光る

**第2話「記憶という代償」**
- ルナから世界の基本情報を教わる。記憶術式の存在を知り衝撃を受ける
- 小さな村にたどり着く。村人は魔法を恐れている
- 蓮がグリモワールの声（アッシュ）を初めて聞く
- 引き: 村に正体不明の武装集団が接近

**第3話「剣と眼帯の男」**
- 武装集団は野盗。村を襲撃する
- 蓮とルナが応戦するも苦戦。そこにガルドが現れ、圧倒的な剣技で野盗を撃退
- ガルドは「忘れた仲間を探している」と語る。蓮のグリモワールに興味を示す
- 引き: ガルドが「その本、ヴェルディアの……いや、なんでもない」と言いかける

**第4話「クレセント港の影」**
- 3人でクレセント港へ。情報を集めるため
- ミレーヌと出会う。記憶の売買の存在を知る
- ミレーヌは蓮のグリモワールを見て動揺するが、平静を装う
- 引き: ミレーヌの後を灰の回廊の追手が尾行している

**第5話「灰の回廊」**
- 灰の回廊の追手がミレーヌを襲撃。蓮たちが巻き込まれる
- 初めてのパーティ戦闘。蓮がグリモワールで支援魔法を使い、ガルドとルナが前衛
- ミレーヌの過去が一部明かされる。追手を撃退
- ミレーヌが仲間に加わる（「借りを返すだけよ」）
- 引き: 追手の一人が「グリモワールの持ち主……灰の塔に報告しなければ」と呟く

### 第二幕: 真実への旅（第6話〜第13話）

**第6話「記憶管理局」**
- レグナムに到着。記憶管理局のゼノに面会
- ゼノはグリモワールに興味を示し、ヴェルディアについての情報を少しだけ教える
- 蓮が自分の記憶の断片を夢で見る（現代日本の映像）
- 引き: ゼノが密かに「あの子がついに現れたか」と独白

**第7話「忘れられた街」**
- ゼノの情報をもとに、ヴェルディア跡地を目指す旅へ
- 道中、蓮が魔法の練習。記憶を失う恐怖と向き合う
- 仲間との日常シーン。キャラクターの関係性を深める
- 引き: 霧の荒野に入ると、ルナの紋章が反応し始める

**第8話「霧の中の記憶」**
- 霧の荒野を進む。霧が幻覚を見せる（各キャラの失われた記憶の断片）
- ガルドが「思い出せない仲間」の幻を見て動揺
- 蓮が幻覚の中で「日本での生活」を垣間見るが、すぐに消える
- 引き: 霧の奥に、巨大な城壁の影が浮かぶ

**第9話「忘却の王国の門」**
- ヴェルディアの廃墟に到着。300年の時が止まったような光景
- ルナが無意識に城壁の隠し通路を見つける（身体が覚えている）
- 内部に残された記録から、禁呪の真実の一端が見えてくる
- 引き: 城の奥から、微かに声が聞こえる

**第10話「王の残響」（中間ターニングポイント）**
- 城の奥で「ヴェルディア最後の王」の記憶の残響（ホログラムのようなもの）と遭遇
- 王が語る: 禁呪は「記憶を世界から消す」のではなく「別の場所に移す」ものだった
- 灰のグリモワールはその「記憶の受け皿」——全ての忘れられた記憶が格納されている
- ルナが王族の末裔であることが判明
- 引き: アッシュが蓮に警告「記憶を解放すれば、お前は消える」

**第11話「選択の重み」**
- 蓮が衝撃の事実に向き合う。記憶を解放すれば自分が消える可能性
- 仲間たちとの対話。ガルドが「逃げてもいい」と言い、ルナが「一緒に考える」と言う
- ミレーヌが灰の回廊の知識を使い、別の方法がないか模索
- 引き: 灰の回廊の大規模部隊がヴェルディア跡地に接近

**第12話「追手」**
- 灰の回廊がヴェルディアに到着。グリモワールを狙う
- 記憶の力を商業的に独占しようとする回廊の野望が判明
- 激しい戦闘。蓮たちは撤退を余儀なくされる
- 引き: ミレーヌが回廊の元上司と再会。「戻ってこい、ミレーヌ。お前の居場所はここだ」

**第13話「裏切りと信頼」**
- ミレーヌが回廊に戻るか葛藤。一度は仲間を裏切りかけるが、最後に蓮たちを選ぶ
- 回廊の内部情報を使って包囲を突破
- 蓮が初めて「感情の記憶」を使った強力な魔法を発動（代償: ルナと初めて会った時の感情の記憶）
- 引き: 蓮がルナを見て「……君は、誰だっけ」と呟いてしまう

### 第三幕: 決戦と再生（第14話〜第20話）

**第14話「欠けた記憶」**
- 蓮がルナとの出会いの感情を失ったことの影響。知識はあるが感情が伴わない
- ルナが初めて感情を見せる（悲しみ）——泣かずに、手の震え・視線の揺れ・「……飲んで」の二語で表現
- チームの士気低下。ガルドが「忘れたからって終わりじゃない、積み直せばいい」と自身の経験を重ねて皆をまとめる
- 蓮がアッシュとの対話で「味方でも敵でもない。ただ、自分の中にいる」という認識に到達し、「感情ではなく、意志として」歩き出す決断をする
- グリモワール六行目の輪郭が前より鮮明に——感情を失った代償と浮上の逆説的関係
- 引き: 灰色の鳥が羊皮紙の封書を運ぶ。グリモワールと同じ書体で「灰より呼ぶ、名を忘れた器よ——還るべき炉を、我らは灯し続けている」。差出人不明。アッシュが微かに反応

**第15話「灰の塔」**
- 五日の山旅を経て灰の塔に到着。ルナの紋章と塔の門の紋様が呼応し門が開く
- 盲目の守護者カインと対面。封書の発信者であり、「還るべき炉」とは記憶が灰になることを拒む塔そのものだと明かす（ただし多義性を維持）
- カインが記憶を「消費する」のではなく「共有する」術式——既存4分類の外にある「原初の術式」（記憶共有術式）の存在を示す
- カインの語りでアッシュが三百年前に自らを書に刻んだ術者の残響であることが部分的に判明。アッシュは尊大さの消えた「昔の話だ」の一言だけを返す——パターン化を崩す転換点
- ガルドが「三百年前……ヴェルディアの時代か」と呟き、ヴェルディア騎士仮説への布石が進行
- 最上階で暁のグリモワール（夜明け前の空の色の表紙）を発見するが起動条件は明かされず。六行目が浮かびかけて止まる
- 引き: 二冊のグリモワール——灰色と夜明けの色——が並ぶ。蓮は「何かが足りない」と感じている

**第16話「二冊目の書」**
- カインが暁のグリモワールの起動条件を明かす:「自らの意思で記憶を手放す者」の涙
- 起動条件の重みに蓮たちが向き合う。誰が、どのように条件を満たすのか
- カインの動機・蓮を呼んだ真意がさらに示唆される
- 引き: 灰の回廊が塔を包囲

**第17話「包囲戦」**
- 灰の塔での攻防戦。仲間たちの見せ場
- ガルドが大技を使うために「最後の仲間の記憶」を使おうとするが、蓮が止める
- 蓮が新たな戦い方——記憶を共有する術式（原初の術式）で仲間を強化
- 引き: 回廊のボスが直接乗り込んでくる

**第18話「記憶の洪水」**
- 回廊ボスとの決戦。グリモワールに格納された大量の記憶が暴走しかける
- 蓮が記憶の洪水に飲まれ、ヴェルディアの歴史を追体験
- 真実の全貌: 蓮はヴェルディアの王が「未来から呼んだ救済者」だった
- 引き: 暴走する記憶を止めるには、蓮が全ての記憶を解放する必要がある

**第19話「忘れない」**
- 最終決戦。蓮が自らの記憶と向き合う
- 仲間たちがそれぞれの「覚悟」を見せる
- 蓮が記憶を解放する決意。しかし仲間との記憶だけは手放さない方法を見つける
- ルナが涙を流し、暁のグリモワールが起動

**第20話「灰と暁」**
- 暁のグリモワールの力で、ヴェルディアの記憶が世界に戻り始める
- 蓮は多くの記憶を失うが、仲間との絆は残る
- 新たな旅の始まり。蓮は「新しい記憶を作る」ことを選ぶ
- エピローグ: ヴェルディアの復興が始まる。蓮が朝日の中で微笑む
- 引き（第二部へ）: 蓮の元に「三冊目のグリモワール」の噂が届く
