# 第6話 創作方針

## エピソードタイトル（仮）
「記憶管理局」

## あらすじ概要
クレセント港を発った四人はレグナム連合の首都レグナムに到着する。ミレーヌの導きで記憶管理局の局長ゼノと面会した蓮は、灰のグリモワールの異常性を指摘され、ヴェルディア王国に関する断片的な情報を得る。その夜、蓮は夢の中で現代日本の光景——教室、夕焼け、呼びかける誰かの声——を見るが、目覚めとともに輪郭が溶けていく。一方ゼノは、蓮たちが去った執務室で独り「あの子がついに現れたか」と呟き、物語は新たな局面へと動き出す。

## シーン構成
1. **オープニング**: レグナム到着。大陸最大の都市の活気と規模に圧倒される蓮。四人パーティとしての最初の旅路を簡潔に描き、ミレーヌが「この街なら情報が集まる」とレグナムを目的地に選んだ経緯を示す。石壁に囲まれた都市の威容、喧騒の音、焼き菓子や香辛料の匂いなど五感描写でレグナムを印象づける。
2. **展開1**: 記憶管理局への訪問。ミレーヌの裏の人脈を使いゼノとの面会を取り付ける。ゼノは穏やかだが底知れない人物として描く。蓮のグリモワールを見たゼノが「これは……」と目を見開き、興味と警戒を同時に示す。ゼノはヴェルディア王国について「公式には存在しなかったことになっている」としつつ、禁呪によって国ごと忘却された可能性を匂わせる。情報は断片的に——全貌は明かさない。ガルドが「ヴェルディア」の名に反応するが、口を噤む描写を入れる。
3. **展開2**: 宿での夜。四人の日常シーンを通じて関係性を深める。ガルドがミレーヌに記憶商人の過去について不器用に歩み寄るやりとり、ルナが窓辺で月を眺める静かな場面など。蓮はグリモワールの声（アッシュ）から「ゼノを信用するな」と警告を受けるが、理由は語られない。就寝後、蓮は夢を見る。
4. **クライマックス/引き**: 蓮の夢——断片的な現代日本の映像。蛍光灯の白い光、机の並ぶ教室、窓の外の夕焼け、自分を呼ぶ声。だが声の主の顔は見えず、名前も聞き取れない。目覚めた蓮は「あれは……俺の記憶だったのか」と震える手でグリモワールに触れる。場面転換し、記憶管理局の執務室。ゼノが蓮たちの去った椅子を見つめ、「あの子がついに現れたか……三百年、待ったのだからな」と独白する。穏やかな仮面の下に、計り知れない思惑が垣間見える。

## キャラクター焦点
- メイン: **蓮** — 新たな都市で情報を得る主体として動く。夢の描写を通じ、失われた記憶への渇望と恐怖の両面を見せる。第5話の戦闘で仲間に指示を出した成長を自然に引き継ぎ、やや能動的になった蓮を描く。
- サブ1: **ゼノ** — 初登場。穏やかな語り口と底知れない含みを持つ人物として印象づける。蓮に対する態度が「協力的だが何かを隠している」ことを読者に感じさせる。
- サブ2: **ミレーヌ** — レグナムでの案内役として有能さを見せる。灰の回廊を離れても情報網を持つ実力者としての一面を描く。
- サブ3: **ガルド** — ヴェルディアの名への反応で、彼自身の過去との繋がりを匂わせる。ミレーヌとの間にある微妙な緊張と歩み寄りを描く。
- サブ4: **ルナ** — 言葉少ないが、蓮の夢の動揺を察するような振る舞いで存在感を出す。

## 伏線・布石
- 回収する伏線:
  - 第2話で蓮が「灰の塔」に既視感を覚えた件 → ゼノの口からグリモワールと灰の塔の関連が示唆される
  - 第3話でガルドが「ヴェルディアの……」と言いかけた件 → ゼノがヴェルディアに言及した際のガルドの反応で再び暗示
  - 第5話ラストで追手が「灰の塔に報告」と言った件 → ゼノが灰の塔の存在に触れることで、追手の報告先の意味が広がる
- 新たに張る伏線:
  - ゼノの「三百年待った」独白 → ゼノの正体と蓮が呼ばれた理由への伏線
  - アッシュの「ゼノを信用するな」という警告 → ゼノとグリモワール（アッシュ）の間にある因縁を示唆
  - 蓮の夢に出てきた「呼びかける声」 → 現代日本に蓮を待つ存在がいることを匂わせる

## 注意事項
- ゼノの情報提供は断片的に留めること。ヴェルディアの全貌や蓮の正体に関わる核心は明かさない。読者に「もっと知りたい」と思わせる匙加減を意識する。
- レグナムの都市描写は冒頭で手短に。世界観の広がりを感じさせつつ、説明過多にならないよう五感描写を中心に。
- 日常シーン（展開2）は四人パーティの関係性を深める大切な場面だが、テンポが落ちすぎないよう注意。会話のキャッチボールで軽快さを保つ。
- 蓮の夢の描写は断片的・感覚的に。具体的すぎると謎が薄れる。映像のフラッシュバックのように短い描写を重ねる手法で。
- ゼノの独白は最終段落に置き、読者の視点を蓮から切り離す唯一の例外的描写とする。三人称一元視点からの逸脱だが、引きの効果を優先する。writing-guide.md の視点ルール（蓮の視点のみ）からの意図的な逸脱であることを作者に伝え、最小限に抑えるよう指示する。
- 第5話で蓮がパンの記憶を失ったことを、何気ない場面で想起させる描写を入れると良い（例: 宿で食事する際、パンを口にしても何も感じない、など）。
